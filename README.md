# Keychn

### develop
```bash
$ git clone git@github.com:ASJvine/vod-platform.git
$ cd vod-platform
$ yarn
$ yarn start # starts express server on port 8000
# different terminal
$ yarn dev:wds # starts webpack-dev-server on port 7000
```

Open your browser at `http://localhost:8000` and hack away :)
